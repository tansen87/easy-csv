# xan transpose

```txt
转置给定的 CSV 文件.

此文件:

A,B
C,D

将变成:

A,C
B,D

Usage:
    xan transpose [options] [<input>]
    xan t [options] [<input>]
    xan transpose --help
    xan t --help

Common options:
    -h, --help             显示帮助
    -o, --output <file>    将输出写入<file>而非stdout
    -d, --delimiter <arg>  读取 CSV 数据时的字段分隔符,必须是单个字符
```
